Sistema de control de acceso con identificación facial.
El desarrollo de un sitio web esta dirigió hacia el personal de una industria siderúrgica, tiene como objetivo llevar el control de entradas, salidas y personal que se encuentre dentro de la planta; los resultados de tener un sitio web ofrecen:
•	Registrar entrada o salida del personal automáticamente utilizando herramientas de identificación biométricas.
•	Permitir la gestión del personal dentro de la planta.
•	Visualizar o descargar el historial del sistema.
•	Realizar búsquedas de registros por semana, mes o por personal.
•	Se presenta una estadística de uso, contiene los métodos de registros de entrada, horarios y duración.
•	Se presenta un resumen del registro diario o actual.
•	Permite el conocimiento del estado del sistema, cámara, base de datos, seguridad y última actualización realizada.
El lenguaje de programación para el diseño del backend es C#, con el uso del framework NET. 8, usando el IDE Microsoft Visual Studio, también se anexa el lenguaje SQL para el desarrollo de consultas con el gestor Microsoft SQL Server. Para el diseño del fronted se utiliza el lenguaje HTML (lenguaje de marcado de hipertexto) para presentar el contenido del sitio web y CSS para obtener una página intuitiva y visualmente atractiva hacia el usuario.

Este sitio web contiene 5 vistas:
- Login
- Control de Accesos
- Entry 
- Exit
- History
